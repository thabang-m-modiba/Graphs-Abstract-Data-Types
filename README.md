# Graphs
## Definition of a Graph.
<p>
  Not your usual graph 😄. A Graph is a way of representing relationships that exis between pairs of objects. It is a non-linear data structure that represents relationships between objects (the vertices or nodes of the graph).
</p>
<p>
  A Graph data structure consists of a finite set of vertices(also called nodes or points), together with a set of edges(also called links or ,ines) that connect these vertices.
</p>
